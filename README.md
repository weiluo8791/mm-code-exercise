# Install instructions

Clone repository

    $ git clone git@github.com:weiluo8791/mm-code-exercise.git

Install sass

    $ npm install -g sass

Compile scss to css
    
    $ sass mm-code-exercise.scss global-styles/mm-code-exercise.css

Just load [code-exercise.html](/code-exercise.html) in the browser
